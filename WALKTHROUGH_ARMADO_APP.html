Ingeniería de Software

Arquitectura del Sistema

La aplicación implementa un patrón cliente-servidor optimizado para entornos de baja latencia con integraciones distribuidas:

Cliente Web (SPA)
Vite + React.js, Three.js (3D engine), Leaflet maps
Servidor Express
Node.js, middleware de seguridad y enrutamiento
Caché & Base de Datos
Caché en memoria del servidor + DB en JSON
APIs Externas
Gemini 2.5 API (VigIA), Open-Meteo API

Despliegue y Rendering 3D a demanda: Para agilizar el rendimiento inicial, el motor Three.js para la modulación visual de estructuras en 3D solo se carga en memoria tras la interacción del usuario mediante un interruptor de activación explícito.

Lógica de Servidor

Motor de Datos y Caché de Memoria

Para garantizar respuestas inmediatas a la consola comercial y en particular al Asistente IA VigIA, implementamos una capa de **procesamiento condensado en caché de memoria**:

Carga Eficiente de Datos

El backend procesa y unifica múltiples tablas al iniciar (Clientes, OTs, Stock de Accesorios, Estructuras y Ventas Históricas). El resumen se almacena en memoria, reduciendo el tiempo de consulta a solo ~10ms en lugar de analizar archivos xlsx recurrentes.

Invalidación y Stock Dinámico

Autocleaning: La caché se invalida y reconstruye instantáneamente tras cualquier acción de escritura (POST/PUT/DELETE).
Stock de Arcos: El balance ya no lee la columna estática de estructuras, sino que cuenta arcos físicos en base_arco y resta reservas de OTs activas.

Seguridad e Higiene

Integración Climática Preventiva

Las condiciones climáticas adversas en la zona geográfica de destino son uno de los mayores riesgos para el armado de estructuras. El sistema incorpora un detector de alertas inteligentes:

Geolocalización en Tiempo Real

Al presionar el botón "Análisis Climático" en cualquier Orden de Trabajo, el sistema lee las coordenadas asignadas de la obra y conecta con el API meteorológico para extraer previsiones de viento, lluvia y ráfagas para el día corriente y los siguientes 7 días.

Evaluación por IA Gemini

La información climática cruda es procesada por el modelo de IA. Si se detectan vientos sostenidos superiores a 35 km/h o tormentas eléctricas severas, el sistema emite alertas de suspensión de armado o recomendaciones de anclaje de seguridad.

Inteligencia de Negocios

VigIA: Asistente de Consultas Comerciales

VigIA representa el núcleo cognitivo de la consola de Gerencia. Es un agente de inteligencia artificial entrenado para interactuar sobre la base de datos viva preprocesada del ERP:

  • Análisis de Proyecciones de Ventas: El usuario puede formular consultas en lenguaje natural respecto a la estimación de volumen de metros cuadrados de carpa a alquilar para meses de 2026 comparándolo con los mismos periodos históricos de 2025.
  • Detección de Recurrencia y Estacionalidad: Identificación automatizada de perfiles de clientes recurrentes (aquellos que arriendan estructuras de forma recurrente todos los meses de Julio o Agosto para exposiciones o campeonatos de invierno).
  • Cálculo Dinámico de Disponibilidades: Responde de forma precisa balances y existencias cruzando arcos físicos en inventario y reservaciones temporales por fecha en OTs activas.

3. GPS Flota & ETA

Conexión con los dispositivos de posicionamiento a bordo para calcular tiempos estimados de arriba a obra y validar arribos de entrega.
